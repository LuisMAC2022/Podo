# Podo

Estructura estática para el consultorio de Enfermería y Podología Soberanes. Los recursos se separan por tipo para facilitar el mantenimiento, la reutilización de estilos y el estudio por secciones.

## Estructura de carpetas
- `pages/`: documentos HTML por pantalla.
  - `index.html`: página principal con servicios, preguntas frecuentes y contacto.
  - `formulario.html`: historia clínica podológica.
- `assets/css/`: hojas de estilo separadas por página.
  - `home.css`: estilos de la página principal.
  - `form.css`: estilos del formulario clínico.
- `assets/images/`: imágenes y logotipos utilizados en las páginas.

## Uso
Abre cualquiera de los archivos dentro de `pages/` en el navegador. Las rutas relativas ya apuntan a los activos dentro de `assets/`.
